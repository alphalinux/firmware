This directory contains a CD-ROM iso image for updating the
firmware on an API NetWorks CS20 or UP2000/UP2000+ system.

The preferred way of updating the firmware on an API NetWorks
CS20 is via the Linux Flash Upgrade Utility, this CS image is
being provides as a convenience for users who wish to upgrade
via LFU.

To use this image, burn it to a CD-ROM and follow the directions in
the readme.1st file.
